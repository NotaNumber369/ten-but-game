## ten-but-game / TENBUT
embarassingly simple game so we teach ourselves some networking

# main idea
a simple 2-player web game where one player holds a hidden rating (1–10) and the other has to match it

## How it works

- player A gets a hidden number (1–10)
- player B writes a condition (e.g. "she's perfect, 10/10 but she hates your family")
- player B guesses the final rating (e.g. if the potential hypotehtical 'she' - if it's a dealbreaker that she doesn't like his family, then she's not a perfect 10/10 but perhaps a 3/10 worth of a person. otherwise, if person A doesn't care much and still finds not much of a problem with the 'she' -> rating stays high)
- the correct number is revealed
- score is compared
